Edit Distance (Levenshtein)
Consider two strings str1 and str2, we need to find the minimum number of operations required to convert str1 to str2.
You have the following 3 operations permitted on a word:

  • Insert a character
  • Delete a character
  • Replace a character
  str1 = "reactive", str2 = "creative"
  Output:  2
  We can convert str1 into str2 by inserting 'c' 
  at starting and deleting c at index 3(str1[3]).
